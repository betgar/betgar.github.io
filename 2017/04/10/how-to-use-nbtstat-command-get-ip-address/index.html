nbtstat计算机名称和MAC互转

背景

因为个人部署了一个小博客，发布一些经验总结方便自己（顺便方便一下大家）。因为个人的电脑IP地址是动态的，所以经常导致之前分享的地址不能使用，所以经过研究，发下了一个一劳永逸的好办法，唯一要求只需要你记得计算机名称(computer name)。

如果电脑处于休眠状态还需要你记得MAC地址（最好记住这个）。
