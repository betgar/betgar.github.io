小结

step已经完成了一个完整的控件例子的展示，从create控件建立控件到destory控件，整个生命周期的管理。

一个控件最终要的就是几个部分：

  1. 初始化渲染(create)
  2. 控件的事件(event)
  3. 控件的销毁(destroy)
  4. 控件的扩展性(extend)
  5. 可配置性(configurable)：通过option来动态改变控件的状态。
