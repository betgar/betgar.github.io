深度操作系统deepin 15.11 安装微信新版本

微信客户端在深度系统15.11版本太低，不给登录的解决办法

环境信息

安装步骤

如果已经安装并使用过，可以调过步骤1、2、

  1. 先安装深度商店的微信客户端

sudo apt install deepin.com.wechat

  1. 启动一次刚刚安装的微信客户端

  2. 现在新版本的微信客户端

    https://pc.weixin.qq.com/

  3. 使用命令安装新版微信

  4. 然后等着图形化的安装界面，一路下一步，就可以了

注意：

问题：测试发现微信不支持双显示器的情况，双显示器情况下，会看不到微信的界面

解决方法：启动微信时，修改显示模式为复制，启动之后，再修改回来

  1. 设置 > 显示> 复制
  2. 启动微信，看到主界面
  3. 设置 > 显示 > 扩展

感谢网友提供的方法：https://bbs.deepin.org/forum.php?mod=viewthread&tid=189623&page=1#pid666765
