nrm管理下载源

nrm 是为了解决切换 npm 源的问题，因为如果直接通过 npm 手动设置 registry 非常麻烦，并且不容易记忆，因此，才有了 nrm 的出现。npm的官方源对广大中国开发者来说较卡，因此推荐 cnpm 或者 taobao。
