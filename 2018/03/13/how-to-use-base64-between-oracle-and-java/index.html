Oracle

UTL_RAW：处理二进制的数据包。

UTL_ENCODE：处理编码包。

  • UTL_ENCODE.BASE64_ENCODE使用

    步骤：

    1. 先用UTL_RAW.CAST_TO_RAW把varchar2转换为raw类型.
    2. 使用UTL_ENCODE.BASE64_ENCODE编码.

    3. 使用UTL_RAW.CAST_TO_VARCHAR2从raw(二进制类型)转换为varchar2类型转换.

      注意： raw类型也可以进行传递，但是Java中要使用Byte进行解析。
